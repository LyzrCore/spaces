# IT Service Desk

AI-powered Incident Management System demonstrating the IT Incident Response blueprint.

## Features

- **Dashboard**: Overview of incident metrics and recent activity
- **Incident List**: View all incidents with status and severity
- **Report Incident**: Submit new incidents for AI-powered analysis

## Powered By

This demo is powered by Lyzr's IT Incident Response blueprint, featuring:
- Incident Coordinator (Manager Agent)
- System Monitoring Agent
- Alert Triage Agent
- Root Cause Analysis Agent
- Escalation Coordinator Agent

[View Blueprint in Lyzr Studio](https://studio.lyzr.ai/blueprints)
